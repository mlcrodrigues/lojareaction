# Webstore React

## Overview
This project is a responsive webstore built using JavaScript and React. It provides a user-friendly interface for browsing and purchasing products.

## Project Structure
```
webstore-react
├── public
│   └── index.html
├── src
│   ├── components
│   │   └── ProductCard.js
│   ├── pages
│   │   └── Home.js
│   ├── App.js
│   ├── index.js
│   └── styles
│       └── main.css
├── package.json
└── README.md
```

## Features
- Responsive design for optimal viewing on various devices.
- Product listing with individual product details.
- Ability to add products to the shopping cart.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd webstore-react
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage
- Open your browser and go to `http://localhost:3000` to view the webstore.
- Browse through the products and add items to your cart.
